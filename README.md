# Flutter Projects

A collection of Flutter applications developed during Flutter learning by Anubhav.

## Projects

### Expense Tracker
A Flutter app for tracking personal expenses. Features include adding, editing, and deleting expenses, with data persistence using SQLite.

**Location:** `expense_tracker/`

**To run:**
```bash
cd expense_tracker
flutter pub get
flutter run
```

### Periods Tracker
A Flutter app for tracking menstrual cycles. Includes authentication, homepage, and Firebase integration for data storage.

**Location:** `periods/`

**To run:**
```bash
cd periods
flutter pub get
flutter run
```

### TG
A Flutter app with login functionality and main page for Telegram-related contest

**Location:** `tg/`

**To run:**
```bash
cd tg
flutter pub get
flutter run
```

## Requirements
- Flutter SDK
- Dart
- Android Studio or Xcode for mobile development

## Getting Started
1. Clone the repository
2. Navigate to the desired project folder
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app on a connected device or emulator
